Dials and Widgets
This project is designed to be a GUI that represents a car Dashboard.  It will have a number of widgets to indicate various stats that might be seen on a car.

These Include dials to indicate:
Speedometer
RPM
Oil Pressure
Temperature
Odometer
Trip Meter
Electric Charge
Gear

The project can be opened on a mac through lab3.xcodeproj

Documentation for the project can be found in DOCUMENTATION.txt, found in the same directory as this file.
